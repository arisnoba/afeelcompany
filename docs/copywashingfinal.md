## 카피 워싱 패턴 요약

### 반복적으로 워싱이 필요한 표현들

| 원본 패턴 | 개선 방향 | 이유 |
|-----------|-----------|------|
| "완벽한 ~" | 삭제 또는 구체적 묘사로 대체 | 완벽함을 자임하면 신뢰보다 부담감을 줌 |
| "가장 ~한" | 삭제 | 최상급 표현은 검증되기 전까지 과장으로 읽힘 |
| "장악" | "높은 비중", "상위권" 등으로 대체 | 공격적 어감. 패션 PR 톤과 불일치 |
| "임팩트" | "결과", "성과" 등으로 대체 | 외래어 강조 표현보다 담담한 한국어가 더 자연스러움 |
| "품절 대란" | "재고 소진", "완판" | "대란"은 자극적. 사실을 사실로 말하면 충분 |
| "~를 증명합니다" | 삭제하고 사실 서술로 | 자기 증명 선언보다 실제 내용이 설득력 있음 |
| "~가 우리의 차별점입니다" | 삭제 | 차별점은 읽는 사람이 느끼는 것, 선언이 아님 |
| "우연한 노출은 없습니다" | 문맥에 따라 재구성 또는 삭제 | 방어적 톤. 긍정의 형태로 재표현 |
