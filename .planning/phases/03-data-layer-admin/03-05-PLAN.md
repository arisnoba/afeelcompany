   - verify admin session
   - load queue row + related portfolio item image URL
   - call `createMediaContainer(imageUrl, caption)` then `publishMediaContainer(container.id)`
   - on success, update `status = 'published'`, `published_at = NOW()`, `external_post_id`
   - on failure, update `status = 'failed'`
   - return `{ success: true, data: { externalPostId } }` or `{ success: false, error }`

Use parameterized SQL only. Do not accept access tokens from request body.
