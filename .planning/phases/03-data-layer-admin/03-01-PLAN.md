  <action>
Create `src/lib/auth.ts` with these exact exports and behaviors:

1. `export const ADMIN_SESSION_COOKIE = 'afeel_admin_session'`
2. `export function verifyAdminPassword(input: string): boolean`
   - read `process.env.ADMIN_PASSWORD`
   - return `false` if missing
   - compare the provided password to `ADMIN_PASSWORD`
3. `export async function createAdminSession(): Promise<void>`
   - create a token shaped as `{expiresAt}.{signature}`
   - `expiresAt` is a unix ms timestamp 7 days ahead
   - signature is an HMAC-SHA256 of `expiresAt` using `ADMIN_PASSWORD`
   - store it in cookie `afeel_admin_session`
   - cookie options must be: `httpOnly: true`, `sameSite: 'lax'`, `secure: process.env.NODE_ENV === 'production'`, `path: '/'`, `maxAge: 60 * 60 * 24 * 7`
4. `export async function clearAdminSession(): Promise<void>`
   - delete `afeel_admin_session`
5. `export async function isAdminAuthenticated(): Promise<boolean>`
   - read the cookie with async `cookies()`
   - reject missing token, malformed token, expired timestamp, or invalid signature
6. `export async function requireAdminSession(): Promise<void>`
   - call `isAdminAuthenticated()`
   - if false, `redirect('/admin/login')`

Do not use localStorage, client-side secrets, unsigned cookies, or synchronous `cookies()`.
