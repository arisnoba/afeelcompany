  <action>
Create `src/app/pdf-export/_components/AutoPrintOnLoad.tsx` as a client component with this exact behavior:

- Read `print` from `useSearchParams()`
- If `print !== '1'`, do nothing
- Otherwise:
  - wait for `document.fonts.ready`
  - find all `img[data-pdf-image]`
  - call `img.decode()` on each image and ignore decode failures
  - finally call `window.print()`

Create `src/app/pdf-export/_components/PrintToolbar.tsx` as a client or server component that renders:

- one button or anchor labeled `다운로드`
- one secondary button or anchor labeled `인쇄 미리보기`

Both actions must target the brochure route:
- `다운로드` -> `/pdf-export?print=1`
- `인쇄 미리보기` -> `/pdf-export`
